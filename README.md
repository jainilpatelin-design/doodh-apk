# Doodh Delivery Manager APK

## How to Build APK (Automatic - Free)

### Step 1: Go to GitHub
Open https://github.com and sign in (or create free account)

### Step 2: Create New Repository
Click "+" → "New repository" → Name it "doodh-apk" → Click "Create repository"

### Step 3: Upload Files
Click "uploading an existing file" → Drag ALL these files into GitHub → Click "Commit changes"

### Step 4: Wait 2-3 Minutes
GitHub will automatically build your APK!

### Step 5: Download Your APK
Go to: Your repo → Actions tab → Click the latest build → Scroll down to "Artifacts" → Download "doodh-delivery-apk"

### Step 6: Install on Phone
Transfer the APK to your phone → Enable "Install from unknown sources" in Settings → Tap the APK to install

---
App stores all data in phone's internal storage automatically ✓
